-------------------
Nozbe
-------------------
nozbe.com
+ Разделение по проектам (подсчёт времени, файлы, заметки к проекту)
+ связка задач с контекстами
+ присваивание меток
+ мобильный и десктопный веб-интерфейс
+ iphone приложение
+ офлайн режим
+ импорт-экспорт
+ интеграция с твитером
+ комментарии к задаче
+ простой, удобный красивый интерфейс
- неудобные повторяющиеся задачи
- нет визуального или численного отображения общего прогресса
- ограничение на 5 проектов при бесплатном аккаунте
- нет отчётов
- система проекто-ориентированная, а не недельно-ориентированная

-------------------
GTDify
-------------------
gtdify.com
+ бесплатный, но по функцилональности похож на nozbe
+ возможность добавлять плагины
- слабый дизайн
- не дружественный интерфейс - слишком много страниц 
- нет русского языка
- нет отчётов
- проектно-ориентированный
- нет прогресса или рейтинга
- неудобные повторяющиеся задачи

-------------------
Speckle
-------------------
speckleapp.com
+ понятный интерфейс
+ бесплатный
- очень скудный функционал
- задача как строка (без даты, сложности и времени выполнения)
- проекто-ориентированный
- нет повторяющихся задач
- нет русского

-------------------
Evernote
-------------------
http://www.evernote.com
+ удобный и качественный интерфейс
- ориентирован на заметки, но есть задачи
- недостаточно большая функциональность задач
- лграниченная функциональность бесплатной версии

-------------------
Get it done
-------------------
http://getitdoneapp.com/
+ удобный, красивый, понятный интерфейс
+ мобильные приложения
+ понятная концепция (задачи, проекты, сферы, метки)
+ есть проекты, но можно создавать задачи без них
+ разделение на сферы деятельности
- нет понятия недели
- нет рейтинга или прогресса, нет отчётов
- глючное
- медленно работает
- нет русского

-------------------
Springpad
-------------------
http://springpadit.com

+ огромная фнкциональность, кроме задач (списки, события, тревоги, шопинг листы, фильмы,вина, контакты...)
+ мобильные приложения
+ плагин для хрома
+ великолепный дизайн и простота интерфейса
+ социальная сеть
- ориентирован на заметки, но есть задачи
- нет повторяющихся задач
- нет русского
